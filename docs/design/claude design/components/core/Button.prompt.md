Use for any clickable action; pick the variant by emphasis (one primary per view).

```jsx
<Button variant="primary" iconRight={<ArrowIcon />}>Resume session</Button>
<Button variant="ghost">Review notes</Button>
<Button variant="subtle" size="sm">Add question</Button>
```

Variants: `primary` (clay gradient, the one call-to-action) · `secondary` (raised, neutral) · `ghost` (outline, quiet) · `subtle` (clay tint) · `danger`. Sizes: `sm` · `md` · `lg`. Supports `iconLeft` / `iconRight`, `fullWidth`, `disabled`.
